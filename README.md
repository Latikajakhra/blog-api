# Blog API (FastAPI)

This is a simple Blog API built using FastAPI.

## Features
- Create blog
- Get all blogs
- Get single blog
- Delete blog

## Tech Stack
- FastAPI
- SQLAlchemy
- SQLite

## How to run
1. Clone repo
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Run server:
   uvicorn main:app --reload

Visit: http://127.0.0.1:8000/docs
